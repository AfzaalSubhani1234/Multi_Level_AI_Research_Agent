# 🤖 Multi-Agent AI Research System (Gemini API)

A multi-agent AI system built using **Google Gemini API** that performs:
- Research
- Fact Checking
- Summarization
- Report Generation

Each task is handled by a separate AI agent, working in a pipeline workflow.

---

## 🚀 Features

- 🧠 Multi-Agent Architecture
- 🔎 AI Research Agent
- ✅ Fact-Checking Agent
- ✂️ Summarizer Agent
- 📊 Report Generator Agent
- 🔗 Gemini 2.5 Flash Integration
- 💬 CLI-based interaction

---

## 🏗️ System Workflow

```text
User Question
     ↓
Research Agent 
     ↓
Fact Checker Agent 
     ↓
Summarizer Agent 
     ↓
Report Generator Agent 
     ↓
Final Report
